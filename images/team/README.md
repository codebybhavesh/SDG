# Team Member Images

This directory contains images of our team members. Please follow these guidelines when adding new images:

1. **Naming Convention**: Use lowercase with hyphens (e.g., `team-member-1.jpg`)
2. **Image Size**: Recommended size is 600x600 pixels (1:1 aspect ratio)
3. **File Format**: Use JPG or WebP format for photos
4. **Quality**: High quality (80% or higher)
5. **Content**: Should be professional headshots with a clean, consistent background
