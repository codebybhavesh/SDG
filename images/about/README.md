# About Page Images

This directory contains images used on the About Us page of the website.

## Current Images

### Hero Background
- `hero-bg.jpg` - The large background image for the hero section of the About page.
  - Recommended size: 1920x1080px (16:9 aspect ratio)
  - Should be high quality but optimized for web
  - Should have a nature/greenhouse theme that works well with the green overlay

### Team Photos
- Team member photos should be placed in the `/images/team/` directory
- See the README in that directory for more details

## Adding New Images

1. **Naming Convention**:
   - Use lowercase with hyphens (e.g., `our-nursery.jpg`)
   - Be descriptive of the image content

2. **Image Specifications**:
   - **Hero Images**: 1920x1080px (16:9), JPG format, 80% quality
   - **Content Images**: 1200x800px (3:2), JPG or WebP format
   - **Icons/Logos**: SVG or PNG with transparent background

3. **Optimization**:
   - Compress all images before uploading
   - Use WebP format for better performance where supported
   - Ensure file sizes are under 500KB for regular images, under 1MB for hero images

4. **Accessibility**:
   - All images should have appropriate alt text in the HTML
   - Avoid using text in images that conveys important information
